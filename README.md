
# Quản lý source code bằng GitHub

## 1. GitHub là gì?
GitHub là nền tảng lưu trữ và quản lý mã nguồn sử dụng Git.

## 2. Git là gì?
Git là hệ thống quản lý phiên bản phân tán.

## 3. Các khái niệm
- Repository: kho chứa code
- Clone: tải code về máy
- Push: đẩy code lên GitHub
- Pull: lấy code mới về
- Branch: nhánh phát triển
- Fork: sao chép repo

## 4. Các lệnh cơ bản
```bash
git clone link
git add .
git commit -m "update"
git push origin main
